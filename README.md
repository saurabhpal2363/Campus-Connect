CampusConnect -->

1. CampusConnect is a web platform that helps students find teammates, collaborators, and project partners within their campus.
2. Students can share their skills, interests, and project ideas to connect with the right people for hackathons, startups, and academic projects.

The idea behind CampusConnect is simple:

“You grow in the company you keep.”
The people around you shape your learning and opportunities.

CampusConnect makes it easier for students to build the right network inside their campus.

Problem -->

Many students struggle to:
1. Find teammates for hackathons
2. Connect with students having similar skills
3. Discover project collaborators
4. Build a productive peer network

Most colleges rely on WhatsApp groups or random connections, which are inefficient.

CampusConnect solves this by creating a dedicated platform for campus collaboration.

Solution -->

CampusConnect allows students to:
1. Create a profile with skills and interests
2. Post project ideas or skill requirements
3. Discover students with relevant skills
4. Connect and collaborate on projects
5. Share posts and project opportunities

The platform builds a skill-based collaboration network inside the campus.

Features -->

1. Student profile creation
2. Skills and interests selection
3. Feed section with:
    a). Video posts
    b). Project / collaboration posts
4. Skill-based collaboration discovery
5. Teammate search functionality
6. Clean and modern UI for students

Tech Stack -->

Frontend
1. HTML5
2. CSS3
3. JavaScript

Backend & Database
1. Firebase

Services used from Firebase:
1. Firebase Authentication – for student login
2. Cloud Firestore – for storing user profiles and posts

Vision -->

CampusConnect aims to become a collaboration ecosystem for college students, where ideas, skills, and opportunities come together.

Instead of struggling to find the right teammates, students can focus on building projects and learning together.
